# Recall-Signature-Calculator
![サムネイル画像](https://github.com/r-1317/Recall-Signature-Calculator/blob/main/ScreenShot.png?raw=true)
## 概要
都道府県知事・市町村長の解職請求に必要な署名数を計算します。

## 制作した理由
Googleで検索しても見つからなかったから。

## 使い方
有権者数を入力するだけで必要な署名数を計算できます。
